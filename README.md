# Forecasting-retail-store-sales
There's historical sales data provided for 45 stores located in different regions, with each store containing a number of departments. 

### Sales
Historical sales data, which covers to 2010-02-05 to 2012-11-01. Within you will find the following fields:

Store - the store number

Dept - the department number

Date - the week

Weekly_Sales -  sales for the given department in the given store

IsHoliday - whether the week is a special holiday week

The task is to predict sales of each department based on historical data.

The language used is python.
